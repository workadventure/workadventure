## Usage

These audio files are used for sound notifications when starting and stopping recordings in WorkAdventure. They are available in multiple languages to provide a localized user experience.

## File Format

Files follow the following naming convention:
- `{locale}_recording-start.mp3` - Recording start sound
- `{locale}_recording-end.mp3` - Recording end sound

Where `{locale}` corresponds to the ISO 639-1/ISO 3166-1 language code (e.g., `en-US`, `fr-FR`, `es-ES`, etc.)
