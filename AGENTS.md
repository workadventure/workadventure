# AGENTS.md - WorkAdventure Monorepo

WorkAdventure is a TypeScript monorepo (npm workspaces) for building collaborative virtual worlds.

## Projects
- `play/AGENTS.md` (Svelte/Phaser frontend + pusher)
- `back/AGENTS.md` (backend API)
- `map-storage/AGENTS.md` (map storage service)
- `messages/AGENTS.md` (protocol buffers)
- `libs/AGENTS.md` (shared libraries)
- `tests/AGENTS.md` (Playwright E2E tests)
